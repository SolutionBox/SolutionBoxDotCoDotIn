Android and app development

With ever increasing number of AndroidTM powered smartphones and tablets in the markets, developing applications for such devices promises huge career opportunities. There are companies who develop applications for improving business of others and there are individual developers too who are making applications as standalone products like games, task managers etc.An STP rich in programing concepts, aesthetics latest technologies certainly helps you to push in AndroidTM ecosystem.

Module 1:Introduction to Android

Installing Softwares, tools and Android development framework, understading java programing for app development. programing, managing resources, manifest file , creating activities and their inter-linking and lifecycle.

Module 2:User Interface and asthetics

View Hierarchy and different views, widgets, notification, dialogs, menus, dynamic UI using fragments, navigation and aesthetics of app development.

Module 3: Database management

Understading different type of data and their storage, SQLite database, SQL, Schemas and other elements of DBMS

Module 4: Contents and Services

understading content providers and using them in Android application, making custom content providers. creating services, threading and services, service life-cycle, communication between service and activites.

Module 5: Connectivity and web services

Use of bluetooth, wi-fi for p2p connectivity, managing network protocols, Json and XML parsing, use of Facebook, Twitter, Google APIs in applications

Module 6:Graphics and Multimedia

Use of OpenGL graphics, Card Flip animations, use of audio, video, camera, intertial devices, GPS in app development

Module 7: Use of cross-Platform Technologies

Use of other programing language and layout constructs C#, C, HTML5, CSS etc for making application and porting them on different operating platforms( iOS, Android, FireFox, Bada, BlackBerry OS), learning basics of Xmarine, PhoneGap, app engines, game engines

Module 8: Distribution and Monetizing Android Apps

Publishing application on Google Play and other app markets, understanding APKs and API levels, enhancing user engagement, advertising and other monetizing strategies.
